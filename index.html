Simon Says Game

How To Play ..

  1. In these we have four blocks, labeled as 1, 2, 3, 4.
  2. Blocks blink in a series on each level.
  3. Press Any key from keyboard to start the game.
  4. A random box blinks.
  5. Level 1 begins with one block, and each subsequent level adds one in the proper order.
  6. Repeat sequence by clicking blocks in order.
  7. If correct, another block has been addded to the sequence, and the level has been upgraded.
  8. Unless you make mistake,keep doing what you're doing.
  9. Goal: Reach highest level by remembering sequence.
